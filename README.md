## Official Code for "Improving Feasibility in Quantum Approximate Optimization Algorithm for Vehicle Routing via Constraint-Aware Initialization and Hybrid XY-X Mixing"

This repository provides the official open-source implementation of our paper, *"Improving Feasibility in Quantum Approximate Optimization Algorithm for Vehicle Routing via Constraint-Aware Initialization and Hybrid XY-X Mixing"*. We have included an environment configuration file, `requirements.txt`, to help you easily set up the necessary dependencies.

### Installation

You can install all required packages by running:

```bash
pip install -r requirements.txt
```
